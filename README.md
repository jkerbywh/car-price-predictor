# Car Price Predictor

Interactive machine learning web app built with Streamlit and Random Forest Regression.

## Features
- Predicts car prices based on:
  - horsepower
  - engine size
  - curb weight
  - car width
  - city MPG
  - highway MPG

## Model
- Random Forest Regressor
- R² Score: ~0.955

## Run Locally

```bash
streamlit run app.py
```
